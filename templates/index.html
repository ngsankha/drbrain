Test a subject

You can upload a network file generated from a fMRI scan of a subject. We will process the network and run it through various tests to test for autism in the subject. Only network data in form adjacency matrix of CSV format is accepted.

Results on available data

We have the COBRE's dataset of fMRI scans available in the database. Using that we can run a series of tests for autism in those subjects and consequently predict the results on any other subject.
You can run the tests and analyis again to go through the summary of results for this dataset.
